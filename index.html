Daily Tasks (Ordered List - OL)

This list uses an **ordered list** (`

    `) to present steps in a sequence.

    1. Start the day with a healthy breakfast.
    2. Review and plan coding tasks for the morning session.
    3. Participate in the team stand-up meeting.
    4. Work on implementing new software features.
    5. Take a short lunch and mental break.
    6. Review colleagues' code submissions (Code Review).
